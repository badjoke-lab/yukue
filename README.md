# ゆくえシリーズ / Yukue Series

ゆくえシリーズは、日本の地域文化、宗教施設、弔いの場所について、現在状態、実施履歴、変化、関係、根拠を記録する観測プロジェクト群です。

The Yukue Series is a set of observational data projects that record current state, occurrence history, change events, relations, and evidence for Japanese local traditions, religious sites, and places of memorialization.

Initial development starts with **祭のゆくえ** and a shared monorepo foundation.
